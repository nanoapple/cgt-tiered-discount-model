# Data Directory

This directory holds generated synthetic data. **No real taxpayer or administrative data is included or required.**

## Data Strategy

- The synthetic data generator produces a realistic population using a 5-dimensional Gaussian copula with four investor archetypes (passive long-term, active accumulator, leveraged speculator, legacy holder).
- Large CSV files are excluded from version control to keep the repository lightweight.
- For reproducibility, the random seed is fixed at 2026 in `run_all.R`.
